To run the lexer, ply has to be installed first as ply is required to be imported in the lexer.
Tokens added:
1) Most of the important keywords in Perl language
2) Different kind s of operators like logical,arithmetic etc. 
3) Various special characters used in Perl
4) Three kinds of variable used in Perl - Scalar, Array, Hash
5) Comments
6) String and Number
7) Identifiers (checked to ensure that its name does not match with the names of reserved tokens i.e. keywords)

Whitespaces adn newlines are ignored.

Each lines of input programs are checked for tokens and tokens are printed just after that.
